# augment-atm
augment云端同步
# 第一次使用请按如下步骤

1. **退出** VSCode 里原来的 **augment** 账号  
2. **卸载** VSCode 里原来的 **augment 插件**，然后 **重启 VSCode**  
3. 将这里下载的 **`.vsix` 文件** 拖拽到插件中心安装，安装完成后 **关闭 VSCode**  
4. **申请账号** 并登录，登录成功后会自动获取 **augment 的 token**  
5. 选择 **使用 VSCode 打开**，即可成功使用
